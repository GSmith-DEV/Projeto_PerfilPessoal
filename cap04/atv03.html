2. Ranking de Tecnologias

  1. HTML

    Linguagem de marcação que atua como base estrutural de qualquer página web.

  2. CSS

    Responsável pela aparência visual da interface.

  3. JavaScript

    Permite interatividade e comportamento dinâmico quando utilizado no HTML.

  4. React

    Biblioteca muito usada atualmente para interfaces modernas.

  5. Node.js

    Permite usar JavaScript no backend, facilitando o estudo da linguagem.

3. Glossário Técnico

HTML
Linguagem de marcação usada para estruturar páginas.
CSS
Folhas de estilo usadas para definir aparência visual e layout dos elementos.
JavaScript
Linguagem de programação da web.
Permite criar interatividade e dinamismo.
DOM
Representação da estrutura de uma página.
Permite manipular elementos com scripts.
API
Interface de Programação
Conjunto de regras para comunicação entre sistemas.
URL
Endereço Web
Localização de um recurso na internet.
HTTP
Protocolo de transferência de dados da web.
DNS
Sistema que traduz nomes de domínio em IP.
